## install

```bash
curl -fsSL https://roxzitywolf.github.io/ezlinux.sh | bash
```

or via AUR:
```bash
paru -S ezlinux
```

## what it does

- system update + paru install
- full gaming stack (steam, gamemode, mangohud, lutris, wine, proton-ge, gamescope)
- performance tweaks (cpu governor, vm tweaks, ananicy-cpp, zram, io scheduler)
- visual themes (catppuccin mocha, tela icons, bibata cursor)
- kali linux VM setup (virt-manager + qemu)
- extra tools (btop, fastfetch, fish shell, starship, fzf, zoxide)
